## Question
You are given an array of integers where each integer represents the height of a bar. The width of each bar is 1.
Return the area of the largest rectangle that can be formed among the bars.

**Example 1:**

```java
Input: heights = [7,1,7,2,2,4]

Output: 8
```

**Example 2:**

```java
Input: heights = [1,3,7]

Output: 7
```
